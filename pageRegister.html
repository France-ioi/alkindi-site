Comment participer ?

Édition 2015-2016

  • Le premier tour est terminé. Les scores sont accessibles pour les coordinateurs via leur interface et pour les élèves sur epreuve.concours-alkindi.fr.
  • Les 50% meilleurs sont qualifiés au 2e tour (score ≥ 200 points pour les 2des, ≥ 180 points pour les 2des pro).
  • Le second tour se déroulera en ligne du samedi 9 janvier 8h au samedi 23 janvier 20h inclus (heure de Paris), par équipe de 1 à 4 personnes du même établissement.
  • Vous pouvez déjà former vos équipes pour le second tour sur suite.concours-alkindi.fr.
  • La bande dessinée qui révèle l'intrigue du second tour est en ligne !

Comment s'inscrire ?

Les inscriptions pour l'édition 2015-2016 sont closes.

Accès au 2e tour

Élèves qualifiés :

Les élèves qualifiés lors du premier tour doivent se munir de leur code de qualification fourni par un enseignant, puis suivre un certain nombre d'étapes en partant qualification.concours-alkindi.fr et en suivant les instructions.

Le détail des étapes pour ces élèves est décrit dans cette documentation illustrée.

Élèves non qualifiés :

Les élèves non qualifiés peuvent rejoindre une équipe selon les règles décrites dans la page du manuel du concours. Pour cela, ils doivent créer un compte et se connecter sur suite.concours-alkindi.fr, puis saisir un code d'équipe fourni par un élève qualifié qui crée l'équipe.

Le détail des étapes pour ces élèves est décrit dans cette documentation illustrée.
